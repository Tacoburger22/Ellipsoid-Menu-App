# Ellipsoid-Menu-App

Jar file contains three classes: Ellipsoid, EllipsoidList, EllipsoidMenuApp
Run EllipsoidMenuApp to see a menu that allows clients to read ellipsoid information from a .txt 

.txt should be formatted like:

List Name
\nLabel
\nAxis 1 length (a)
\nAxis 2 length (b) 
\nAxis 3 length (c)
\nLabel
\nand so on...
